Craigslist
Used Car Listings

In this project we examined a used car dataset scraped off of craigslist(US) containing 100,000+ records and 10+ fields. This data originally came in quite messy, unorganized, and missing a lot of values. After wrangling with the data mainly in excel, we were able to clean the data in a visually pleasing and structured manner. Once we had a highly usable dataset, we then took our file to Microsoft SQL server where we began to ask questions and discover insights through various queries. Once all the questions were answered, we went to tableau to visualize these findings.
